# 🌿 Plant Disease Detection Using Machine Learning

## 📌 Project Overview

This project was developed as the final project of a **9-Day Machine Learning Workshop**. The objective of this project is to identify different plant diseases from leaf images using Machine Learning algorithms.

The project processes plant leaf images, extracts features, trains classification models, and predicts the disease category of a plant.

---

## 🚀 Features

- Image preprocessing
- Image resizing and normalization
- Label Encoding
- Dataset visualization
- Logistic Regression model
- Support Vector Machine (SVM) model
- Model evaluation using:
  - Accuracy Score
  - Confusion Matrix
  - Classification Report

## ⚙️ Machine Learning Workflow

1. Import required libraries
2. Extract dataset
3. Load training and validation images
4. Preprocess images
5. Resize images
6. Encode labels
7. Train Logistic Regression model
8. Train Support Vector Machine (SVM)
9. Evaluate model performance
10. Predict plant disease
